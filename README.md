# Vocal Walls : L'Empreinte Sonore Locale

> **"La réalité augmentée sonore pour une ville plus humaine."**

## Vision
Contrairement au géofencing publicitaire traditionnel, **Vocal Walls** est une plateforme communautaire et contextuelle. Elle permet de laisser une empreinte sonore persistante dans le monde physique.

## Concept : Le "Street Marketing" Sonore
L'idée est de créer un pèlerinage numérique. Un artiste, un historien ou un habitant laisse une "bulle sonore". Pour l'entendre, il faut **être sur place**.

## Fonctionnalités Clés
1.  **Géofencing H3 (Uber)** : Précision chirurgicale sans vider la batterie.
2.  **Audio 3D (Agora SDK)** : Le son change selon votre position et l'orientation de votre tête.
3.  **Entropie (Decay)** : Les bulles ont une durée de vie. Sans "likes" (recharge), elles s'effacent progressivement. Seule la qualité survit.
4.  **Audio-Guides Premium** : Parcours exclusifs pour le tourisme.

## Stack Technique
- **H3Indexer.js** : Géofencing hexagonal.
- **SpatialAudioEngine.ts** : Intégration Agora SDK.
- **DecayService.py** : Algorithme de gestion de vie des bulles.
