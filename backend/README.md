# Vocal Walls Backend API

## Prérequis
- Node.js 18+

## Installation
```bash
cd backend
npm install
```

## Démarrage
```bash
npm start
```

API par défaut: `http://localhost:4000`

## Variables d'environnement
- `PORT` (optionnel): port HTTP, défaut `4000`
- `DB_PATH` (optionnel): chemin du fichier JSON de persistance

## Tests
```bash
npm test
```
